# Aadya's Personal Website

This is the repository for my personal website, built from scratch using HTML and CSS. It's a simple, clean site meant to showcase my background, skills, and school activities.

## What's Inside
- **About Me:** A quick background on my interests, sports, and clubs (Coding Club, Girls Who Code, and Math Club).
- **Skills Matrix:** A responsive grid showing the tech I work with, complete with smooth hover animations.
- **Sticky Navigation:** A top menu bar that stays fixed as you scroll.
- **Contact Info:** A direct link to reach out via email.

## Tech Stack
- **HTML:** For the page structure and semantic layout.
- **CSS:** For custom styling, flexbox/grid alignments, and button animations.
